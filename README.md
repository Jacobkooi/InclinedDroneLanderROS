# InclinedDroneLanderROS
ROS/C++/Libtorch code for flying a Crazyflie with RL policy networks. This is the C++ extension of https://github.com/Jacobkooi/InclinedDroneLander for completing Real-time experiments with the Pytorch policy network.

This package is used in conjuction with the bebop2 toolbox (https://github.com/hai-zhu/bebop2_toolbox) for the state estimator and the crazyflie_ROS package (https://github.com/whoenig/crazyflie_ros) for the Crazyflie server.
